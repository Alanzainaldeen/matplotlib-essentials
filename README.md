> A comprehensive Jupyter Notebook tutorial demonstrating essential Matplotlib plotting techniques for data visualization in Python.

## 📋 Overview

This repository contains a hands-on Jupyter Notebook that explores the powerful capabilities of **Matplotlib**, Python's premier 2D plotting library. Perfect for beginners and intermediate users looking to enhance their data visualization skills.

## ✨ Features Covered

### 🎨 Styling & Themes
- Explore 30+ built-in styles using `plt.style.available`
- Apply styles: `classic`, `dark_background`, `ggplot`, `seaborn-*`, and more
- Customize plot aesthetics for publication-ready figures

### 📊 Plot Types & Customization
- Line plots with multiple datasets
- Custom markers (`o`, `.`, `x`, `+`, `^`, `<`, `>`, `s`, `d`)
- Line styles (`-`, `--`, `-.`, `:`) and color specifications
- Legend positioning and label management
- Axis limits, labels, titles, and equal aspect ratios

### 🔧 Advanced Techniques
- Figure and axes configuration (`figsize`, `dpi`)
- Scatter plots with dynamic coloring and sizing
- Color maps (`viridis`, `coolwarm`, etc.) and alpha transparency
- Colorbar integration for continuous data representation

### 📈 Real-World Data Visualization
- Integration with **Pandas** for DataFrame plotting
- Loading datasets with **Seaborn** (`iris` dataset example)
- Categorical coloring for multi-class data visualization
